# Обновление навигации

## Изменения в меню

### ✅ Добавлена ссылка "Продукция"
- **Ссылка**: `/categories` 
- **Описание**: Ведет на страницу со всеми продуктами
- **Активное состояние**: Выделяется для всех страниц категорий

### Обновленная навигация:
1. **Главная** → `/`
2. **О нас** → `/about`
3. **Контакты** → `/contacts`
4. **Продукция** → `/categories` ← **НОВОЕ**

## Технические детали

### Логика активного состояния
```typescript
pathname === item.href || (item.href === '/categories' && pathname.startsWith('/categories'))
```

Это означает, что ссылка "Продукция" будет активна для:
- `/categories` (все продукты)
- `/categories/indoor-displays` (внутренние дисплеи)
- `/categories/outdoor-displays` (уличные дисплеи)
- `/categories/rental-displays` (арендные дисплеи)
- И всех других страниц категорий

### Стили активного состояния
- **Фон**: `bg-blue-100`
- **Текст**: `text-blue-700`
- **Переходы**: Плавные анимации при наведении

## Пользовательский опыт

### Навигация по сайту:
1. Пользователь нажимает "Продукция" в меню
2. Переходит на `/categories` (все продукты)
3. Может переключаться между категориями через фильтры
4. Навигация остается активной на всех страницах категорий

### Breadcrumbs эффект:
- Меню показывает активное состояние
- Пользователь понимает, что находится в разделе "Продукция"
- Легко вернуться к другим разделам сайта
